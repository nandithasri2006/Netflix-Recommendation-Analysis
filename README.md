# Netflix-Recommendation-Analysis-and-Visualization
 <h1>Overview</h1>

This project performs data analysis on a Netflix dataset using Python in Jupyter Notebook and presents insights through an interactive dashboard built in Microsoft Power BI. It focuses on understanding trends in content type, genres, release years, and popularity.

<h1>Objectives</h1> 

Clean and preprocess Netflix data

Perform exploratory data analysis (EDA)

Identify patterns and trends

Build an interactive dashboard for visualization

<h1>Tools & Technologies</h1>

Python (pandas, matplotlib)

Jupyter Notebook

Power BI

 <h1>Dashboard Features</h1>

Slicers (Genre, Year, Type)

Bar Chart (Top content)

Scatter Plot (Popularity vs Votes)

Donut Chart (Genre distribution)

Column Chart (Year-wise trends)

Interactive filtering with reset option

<h1>Key Insights</h1>

Movies dominate the dataset compared to TV shows

Certain genres are more frequent

Content production has increased over the years

Popularity is influenced by vote count

<h1> How to Run</h1>

Open the notebook in Jupyter Notebook and run all cells

Open the .pbix file in Microsoft Power BI to view the dashboard
